# PROJECT: IBANKING api

### Requirement

- The latest Node.js possible
- Database MongoDB

### Getting Started

First, we need a database. Use mongodb:

```
Database's name: iBanking
Collections:
  - Fees
  - Users
```

And install all dependencies:

```
npm install
```

Add some data into database:

```
cd utils
```

```
node config.js
```

Run project:

```
cd ..
```

```
npm start
```

Available at: http://localhost:8080
