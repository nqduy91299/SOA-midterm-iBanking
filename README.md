# PROJECT: IBANKING

## **BEFORE USE**

Create Database:

- name database: iBanking
- collections:
  - Fees
  - Users

Install all modules:

```
npm install
```

Add some data into database:

```
cd utils
```

```
node config.js
```

## **USE**

Run project:

```
**npm start**
```

Available at: http://localhost:8080
